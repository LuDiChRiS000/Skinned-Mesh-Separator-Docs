# Skinned Mesh Separator — Documentation

Documentation site for the [Skinned Mesh Separator](https://assetstore.unity.com/packages/tools/utilities/skinned-mesh-separator-346784) Unity Asset Store tool.

Built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/).

## Local Preview

```bash
pip install mkdocs-material
mkdocs serve
```

Then open [http://localhost:8000](http://localhost:8000).

## Deploy

Pushing to `main` automatically deploys to GitHub Pages via the included workflow.
